# Grok API Web UI

Web app to interface with Grok's API. This repo starts with a clean, extensible structure and guidance for consistent development.

## Structure

- docs/INSTRUCTIONS.md: project conventions and structure rules
- src/: application code
- tests/: automated tests
- docker/: container build and compose files
- scripts/: developer utilities

## Quick start

1. Copy .env.example to .env and fill in values.
2. Build and run with Docker Compose.

See docs/INSTRUCTIONS.md for detailed guidance.
